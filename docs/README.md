##・生体代謝の加速による環境の急激な寒冷化

# できる限りのエネルギー消費の限定、軌道上での遮光管理施策を含む、生存圏の維持
# 核兵器等による、太陽への作用干渉によるスケール調和

共形場理論的な表現を借りて理路を述べる。
頂点作用素代数を局所点幾何の最大自由度と理解する。自由度は素数らせんの対数的発展のもと、乗法的にふるまう。代数の群は単純散在群が適用され、局所点構造であるトーラスの回転は非可換である
ふたつの物質の相互作用は、異なるVOA間の対称性の回転である。これは頂点作用素積に相当する。

ひとつの物質を、マチュー群M24の形式で現れる要素点の集まりとする
ふたつの物質間の回転は個々の位相点と、回転の基準系を持つ
24+24+2+1
これは表現上、アルキメデスの素数らせんのに対数らせんに投影されるモンスター部分群Held幾何による51次元表現に従う
